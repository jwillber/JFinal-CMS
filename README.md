# JFinal急速CMS

#### 介绍
JFinal急速CMS，轻量级CMS，企业建站1-3天搞定，后台admintle，前台模板标签，采用JFinal(MVC)，Ehcache(缓存)，Urlrewrite(伪静态) 架构快速搭建个性化独立网站。轻轻松松做站长。

#### 软件架构
MVC:JFinal<br/>
缓存:Ehcache<br/>
伪静态:Urlrewrite<br/>
存储:阿里云OSS<br/>
数据库:Mysql<br/>


#### 安装教程

1.  Maven导入代码
2.  sql文件导入数据库
3.  修改config.properties数据库配置，运行CmsConfig类
